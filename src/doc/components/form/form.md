基础的表单。

`form` 组件内部使用 `form-item` 作为表单项容器。  
使用 `form-item.label` 指定表单项标题。

<preview src="components.form.form" />

使用 `form.label-width` 指定表单项标题宽度。  
使用 `form` 与 `form-input` 中的 `label-align`、`input-align` 属性控制对齐方式。

<preview src="components.form.form-align" />

`form-item` 间可以嵌套，实现多个表单元素在同一表单项内。

<preview src="components.form.form-item" />

`form-item.for` 属性可用于绑定指定表单元素 `id`。

<preview src="components.form.form-id" />
